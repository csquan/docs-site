## 简介与功能概览

Vcity Chain 是基于 DPoS（委托权益证明）的区块链应用，可查看链上 Epoch 信息、出块者与候选人、进行委托投票、管理质押以及参与链上治理。

打开页面后，顶部导航栏有五个入口，点击即可进入对应功能：

| 入口       | 功能说明 |
|------------|----------|
| 首页       | 查看 Epoch 信息、查询奖励、查询区块生产信息 |
| 出块者     | 查看验证者/出块者列表、出块数、质押与奖励、故障状态 |
| 候选人     | 查看候选人列表、注册/撤销候选人、对候选人投票 |
| 我的质押页 | 查看出站/入站投票明细、管理验证者佣金率 |
| 治理页     | 参数治理、故障者恢复治理、查看提案并投票或执行 |

## 首页

点击顶部 **「首页」** 进入。可查看当前 Epoch 信息、按 Epoch 或区块范围查询奖励与区块生产信息。

### 一、Epoch 信息展示

页面加载后会自动请求当前 Epoch 与共识状态，并以卡片形式展示，包括但不限于：**出块数**（blocksProduced）、**Epoch 编号**（epochNumber）、**预期出块数**（epochSize）、**Epoch 开始时间**（epochStartTime）、**剩余区块数**（remainingBlocks）、**共识轮次编号**（currentRound）、**当前出块者地址**（currentDelegate）。部分字段为数值，部分为字符串。

### 二、查询奖励

**输入项**：

- **Epoch 编号起始**：单个 Epoch 时只填此项；查范围时作为起始 Epoch。
- **Epoch 编号结束**：可选。不填则只查「Epoch 编号起始」对应的一个 Epoch；填写则查询该区间内的奖励。
- **验证者地址**：可选。填写后仅展示该验证者相关的奖励记录。

点击 **「查询奖励」** 后，下方会展示奖励明细表格，列包括：记录 ID、Epoch 编号、接受者地址、奖励类型、奖励金额、投票权重、发放时间、交易相关哈希、状态等。

### 三、查询区块生产信息

用于查看指定区块高度区间内，各出块者的出块情况。

**输入项**：**区块起始值**（须大于当前共识切换高度）、**区块结束值**（须大于等于起始值）。点击 **「查询区块生产信息」** 后，会展示模式、区块范围总数，以及表格：出块者地址、区号列表、出块数。

**说明**：金额类数据会按 wei 与 ether 做换算展示；时间类会格式化为 `YYYY-MM-DD HH:mm:ss`。

## 出块者

点击顶部 **「出块者」** 进入。页面展示当前链上验证者（出块者）列表及其质押、出块与奖励、故障状态，并支持进入详情与奖励明细。

### 列表字段说明

| 列名             | 说明 |
|------------------|------|
| 出块者           | 验证者地址，可点击进入详情页 |
| 出块者数量       | 当前 Epoch 出块数 |
| 质押金额(wei)    | 质押数量（wei） |
| 出块奖励金额     | 奖励金额，可点击弹出奖励明细 |
| 已经生效的票数   | 已生效投票数量 |
| 未生效的票数     | 未生效投票数量 |
| 状态             | 有故障 / 没有故障 |
| 故障原因         | 若有故障，显示时间与原因说明 |

### 操作说明

- **进入出块者详情**：点击列表中「出块者」列下的地址，跳转到详情页，可查看该验证者的投票详情、入站/出站质押、奖励历史、佣金率等。
- **查看奖励明细**：点击列表中「出块奖励金额」，会弹出「奖励明细」弹窗，表格列包括：奖励金额、投票权重、奖励所属的 epoch 编号、验证者、出块数、每块奖励、奖励类型、奖励发放时间。

**说明**：列表与详情数据依赖当前 Epoch 信息及链上接口，加载时可能出现短暂 loading。

## 候选人

点击顶部 **「候选人」** 进入。可在此查看链上候选人列表、注册/撤销候选人，以及对候选人进行投票或撤销投票。

### 顶部操作

- **注册成为候选人**：打开注册弹窗，填写信息后提交注册。
- **撤销成为候选人**：打开撤销弹窗，填写注册者地址与私钥后提交撤销。

### 列表字段说明

| 列名     | 说明 |
|----------|------|
| 候选人的地址 | 可点击进入候选人详情页 |
| 候选人名称、官方网站、描述信息 | 注册时填写的展示信息 |
| 保证金金额 | 保证金（wei） |
| 已经生效的票数 / 未生效的票数 | 投票统计 |
| 排名     | 候选人排名 |
| 注册时间、最后投票时间、冻结时间、解冻时间、资金可用时间 | 时间与状态 |
| 质押权重 | 质押数量（wei） |
| 出块奖励金额 | 可点击查看奖励明细弹窗 |
| 状态 / 故障原因 | 与出块者页一致 |
| 操作     | **投票** 按钮，打开投票弹窗 |

### 注册成为候选人

在弹窗中填写：**注册者地址**（必填，合规钱包地址）、**受托人名称**（必填）、**网站地址**与**描述**（选填）、**注册者私钥**（必填，64 位十六进制字符串，不含 `0x` 前缀）、**chainID**（必填，默认 20230826）。点击 **「注册」** 提交；成功后会提示并关闭弹窗。

### 撤销成为候选人

在弹窗中填写 **注册者地址** 与 **注册者私钥**（均为必填），点击 **「撤销」** 提交。

### 投票

1. 在列表中点击某一行的 **「投票」**。
2. 在投票弹窗中填写：**钱包地址**（必填）、**钱包私钥**（选填，用于签名）、**投票金额**（必填，单位 wei，不超过可用余额）。
3. 可查看可用余额、冻结余额、总资产。
4. 点击 **「投票」** 提交投票，或 **「撤销投票」** 撤销对该候选人的投票（撤销时金额由系统处理）。

**说明**：私钥仅用于本地签名，请勿在不安全环境中使用；候选人详情页可通过点击候选人地址进入。

## 我的质押

点击顶部 **「我的质押页」** 进入。用于查看当前连接钱包作为验证者时的 **出站投票**、**入站投票**，以及 **佣金率** 的查看与更新。需要先通过应用连接钱包（如 MetaMask），以便识别「我的」地址。

### 佣金率

- 页面上方展示 **「我的佣金率」** 当前值。
- 点击 **「更新佣金率」** 打开弹窗：**出块者地址**（自动带出当前钱包地址，不可改）、**佣金率**（500–8000，通常为万分比，如 500 表示 5%）、**私钥**（用于签名更新交易）。填写后点击 **「更新」** 提交；成功后刷新页面即可看到新佣金率。

### 出站投票明细

表格列出 **本人对其它验证者的投票**：受托人地址、投票金额（已换算为 ether）、投票开始时间。

### 入站投票明细

表格列出 **他人对本人（当前验证者）的投票**：委托人地址、投票金额、投票开始时间。

**说明**：若未连接钱包或当前地址不是验证者，可能无入站/出站数据；佣金率仅对已注册为验证者的地址有效。

## 治理

点击顶部 **「治理页」** 进入。提供 **参数治理**、**故障者治理** 以及 **查看与参与所有提案**（投票、执行）。顶部为 Tab：参数治理、故障者治理、查看当前所有提案。

### 一、参数治理

用于发起「修改链上可投票参数」的提案。页面会先展示三类可投票参数表：**经济参数**、**治理参数**、**削减参数**（名称、类型、最小值、最大值、参数描述）。

**提交参数提案**：在表单中填写 **参数名称**（必填，从下拉框选择，选择后会提示取值范围与当前值）、**参数新值**（必填，在合法范围内）、**描述**（选填）、**提案者地址**（必填，须为验证者）、**提案者私钥**（必填，64 位十六进制，不含 `0x` 前缀）。点击 **「提交」** 创建提案；成功后可在「查看当前所有提案」中看到新提案。

### 二、故障者治理

用于对因故障被标记的验证者发起「恢复」提案。

**提交恢复提案**：填写 **要恢复的验证者地址**（必填）、**恢复原因**（必填）、**提案描述**（选填）、**提案者地址**（必填，须为验证者）、**提案者私钥**（必填），点击 **「提交」** 创建恢复提案。

### 三、查看当前所有提案

以表格形式列出当前所有活跃提案，列包括：提案唯一标识符（可点击进入详情）、提案对象、类型（参数提案/恢复提案）、当前值、提案建议新值、提案发起者地址、提案生效起始/投票截止区块高度、提案状态（投票中/已通过/已拒绝/已过期）、通过提案最少投票数、提案描述、提案创建时间、当前已收到投票数、**操作**（投票、执行提案）。

**投票**：仅当提案处于投票中且当前区块在投票起止高度之间时，可点击「投票」。在弹窗中填写提案 ID（只读）、投票者地址、投票者私钥，选择「支持」或「反对」后提交。

**执行提案**：当提案已通过且已过投票截止区块（未过期可执行期）时，会显示「执行提案」。在弹窗中填写提案 ID、执行提案者地址、执行提案者私钥，点击「确定」执行。

**说明**：提案状态与区块高度强相关，若「执行提案」不显示，可能是尚未到可执行区块或已过执行期限；详情页通过点击提案 ID 进入。
